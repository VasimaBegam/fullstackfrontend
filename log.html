📋 Exercise Parameter Guide

  • 🔹 Exercise Name: Enter precise nomenclatures (e.g., 'Barbell Bench Press', 'Incline Dumbbell Flyes') for clean structural grouping.
  • 🔹 Actual Sets & Reps: Total completed mechanical sequences. Avoid logging warm-ups; document working sets only.
  • 🔹 Actual Weight (Kg): The true external load configuration. Include the bar weight (typically 20kg for standard Olympic barbells).
  • 🔹 PR Flag Node: Checking this activates visual glowing highlights across data pipelines to preserve your legendary lifting logs.

💡 Performance Console FAQ

  • ❓ How to map drop-sets? Use the POST panel to document separate lines per weight drop or update them seamlessly via PATCH commands.
  • ❓ What does PUT do here? It rewrites the chosen data block entry row completely, ensuring structural validation is preserved entirely.
  • ❓ Can I remove corrupt logs? Yes, clear out faulty or miscalculated logging nodes using the specialized DELETE console mechanism effortlessly.
